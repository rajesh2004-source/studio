# **App Name**: PettyFlow

## Core Features:

- Dashboard: Displays current cash balance, cash inflows, outflows, and recent transactions.
- Vendor Management: Add, update, and delete parties/vendors, assign transactions, and view party-wise transaction history.
- Transaction Module: Add income (cash in) and expense (cash out), choose payment mode, add notes and attachments, and auto-update petty cash balance.
- Reporting: Generate date-wise, category-wise, and party-wise reports, including AI-assisted summaries.
- Smart Categorization: AI tool suggests transaction categories based on description and vendor. The user has to approve the suggested categories.
- Transaction Saving: Store petty cash transactions into a MongoDB database.
- Authentication: JWT-based user login and authentication.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trustworthiness and stability.
- Background color: Light blue (#E8EAF6), a desaturated hue of the primary.
- Accent color: Purple (#7E57C2) for highlights and interactive elements.
- Headline font: 'Space Grotesk', a sans-serif font, which will be paired with 'Inter' for body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear and professional icons for navigation and actions.
- Subtle animations and transitions for a smooth user experience.
